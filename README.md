记账功能的qt软件，具体功能如下


QT展示的页面如下

| 序号 | 记账日期  | 当前总存款金额 | 当月工资 | 定期余额 | 当月开支 | 当月存款 | 备注 |
| ---- | --------- | -------------- | -------- | -------- | -------- | -------- | ---- |
| 1    | 2026/1/29 | x1             | y1       | z1       |          | x1       |      |
| 2    | 2026/2/29 | x2             | y1       | z1       |          | x2       |      |

说明

序号：每次记录的序号，可以随时新增

记账日期： 每个月的29号填写

当前总存款金额 ： 当前这个月的总存款金额 当前身上的所有钱

当月工资：当前这个月发的工资

定期余额： 当前这个月银行的的定期余额钱

当月开支： 当月花了多少钱

当月存款： 当月存了多少钱

备注： 备注


每次记账操作步骤

1打开软件 自动识别软件目前下的 excel表数据，姜表数据展示如上的格式

1 新增一行数据，系统自动获取现在的日期 年月日即可

2 填写 当前总存款金额 和 当月工资 和 定期余额

3 系统自动计算出 当月存款 和 当月开支

3 保存最新信息到excel表里面



当月开支= 上一次记录的 当前总存款金额+当月工资 - 本次填写的当前总存款金额
